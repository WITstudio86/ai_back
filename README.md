# rAi

通过 rust 实现调用智谱 Ai 接口进行回复的 GUI 应用

## cli

- `rai --help` 查看帮助

```shell
Error: Usage: rai <COMMAND>

Commands:
  clear  Clears the history massage
  chat   select the figure
  help   Print this message or the help of the given subcommand(s)

Options:
  -h, --help     Print help
  -V, --version  Print version
```
